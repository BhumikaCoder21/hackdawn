# 🌾 AgriHills – Agricultural Marketplace Platform

AgriHills is a web-based platform designed to connect **farmers and buyers directly**.
It allows users to post and explore agricultural products, making trading easier, faster, and more transparent.

The platform leverages **Firebase** for real-time database management and seamless user interaction.

---

## ✨ Features

* 🌱 **Post Agricultural Products**
  Farmers can list their produce with details like price, quantity, and location.

* 🔍 **Browse Listings**
  Buyers can explore available products posted by farmers.

* ⚡ **Real-Time Updates**
  Data updates instantly using Firebase.

* 📱 **User-Friendly Interface**
  Simple and clean UI for smooth navigation.

* ☁️ **Cloud-Based Backend**
  Uses Firebase for backend services and data storage.

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash id="5o1g6q"
git clone https://github.com/yourusername/agrihills.git
```

### 2️⃣ Navigate to the Project Folder

```bash id="0i1qg8"
cd agrihills
```

### 3️⃣ Install Dependencies

```bash id="d0i9sc"
npm install
```

---

### 4️⃣ Firebase Setup

* Create a Firebase project
* Enable Firestore Database
* Get your Firebase config

Create a `.env` file and add:

```env id="q9l1bp"
REACT_APP_API_KEY=your_api_key
REACT_APP_AUTH_DOMAIN=your_auth_domain
REACT_APP_PROJECT_ID=your_project_id
REACT_APP_STORAGE_BUCKET=your_storage_bucket
REACT_APP_MESSAGING_SENDER_ID=your_sender_id
REACT_APP_APP_ID=your_app_id
```

---

### 5️⃣ Run the Project

```bash id="1x7lwr"
npm start
```

The app will run on:

```id="rzx5ek"
http://localhost:3000
```
